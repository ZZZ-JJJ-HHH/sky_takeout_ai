# 苍穹外卖系统首次提交

## 主要功能
- ✅ 员工登录认证功能
- ✅ JWT令牌生成与验证
- ✅ BCrypt密码加密
- ✅ MySQL数据库集成
- ✅ Redis缓存支持
- ✅ Knife4j API文档
- ✅ Docker环境配置

## 技术栈
- Java 17
- Spring Boot 3.3.3
- MyBatis-Plus 3.5.7
- MySQL 8.0
- Redis 7
- Knife4j 4.3.0
